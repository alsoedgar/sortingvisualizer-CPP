You Real-Time Interactive Sorting Visualizer in C++ using the SDL3 library!

User Controls:
1-5: Switch Algorithms
R: Shuffle the array (Fisher-Yates) and restart
UP/DOWN: Adjust simulation speed in real-time
ESC: Quit

Key Features:
-Implements Bubble, Selection, Insertion, Quick, and Merge Sort, each with unique visualization logic
-Active Highlighting: Bars turn White when sorted, Red/Pink when being compared or swapped, and Neon Green during progress updates
-Progress Bar: A non-regressing bar at the bottom tracks completion
-Synthesized Audio: Generates audio corresponding to the height of the bar being processed
-Algorithm Name & Description
-Time Complexity (Big-O notation)
-Live counters for Comparisons and Swaps
